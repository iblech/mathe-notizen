=== Exaktheitseigenschaften

* In Topoi sind Pushouts von Monos wieder Monos. Das zeigt man unter Verwendung
  injektiver Objekte: Sei

      A `--> B
      |      |
      |      |
      v      v
      C ---> D

  ein Pushout-Diagramm. Wir möchten zeigen, dass C --> D ein Mono ist. Wir
  wählen eine Einbettung C --> E in ein injektives Objekt. Dann wird E zu einem
  Möchtegern-Pushout von B und C über A. Also gibt es genau einen Morphismus
  D --> E, sodass C --> D --> E gleich der Einbettung C --> E ist. Da diese ein
  Mono ist, ist auch C --> D ein Mono.


=== Darstellung von Topoi

* Sei E eine präsentierbare Kategorie (damit meint Joyal "lokal präsentierbare
  Kategorie"). Dann gibt es eine kleine Kategorie A, sodass PSh(A) --> E
  eine Lokalisierung ist.

* Wenn E sogar ein Topos ist, kann man A so wählen, dass es endlich vollständig
  ist.

* Zu jedem Topos E mit genügend vielen Punkten gibt es einen topologischen Raum X
  zusammen mit einem geometrischen Morphismus E --> Sh(X) (der noch tolle
  Eigenschaften hat, wie zum Beispiel, dass der Urbildfunktor noch einen
  Linksadjungierten besitzt), sodass Kohomologie von abelschen Gruppen in E
  zu gewöhnlicher Garbenkohomologie von X wird.
  http://www.academia.edu/9139781/Logical_and_Cohomological_Aspects_of_the_Space_of_Points_of_a_Topos

* Wie allgemein sind Topoi wirklich? Johnstone zeigt in /How general is
  a generalized space?/ (aufbauend auf Barr/Diaconescu/Coste/Joyal--Tierney),
  dass jeder Grothendiecktopos einen zusammenhängenden und lokal
  zusammenhängenden geometrischen Morphismus aus einem Topos von Garben über
  einer Örtlichkeit zulässt.

=== Punkte

* Die Punkte von PSh(C), wobei C eine kleine Kategorie ist, sind genau
  die flachen Funktoren C --> Set. Der Pullbackteil des zu einem solchen
  flachen Funktor F gehörenden geometrischen Morphismus ist

      PSh(C) --> Set,  E |--> E tensor_C F.

  Im Fall, dass C endlich vollständig ist, ist ein Funktor C --> Set genau dann
  flach, wenn er endliche Limiten bewahrt.

  Allgemeiner sind die E-wertigen Punkte von PSh(C) genau die intern-flachen
  Funktoren C --> E.

* Jedes Objekt X von C führt also zum Punkt "Auswertung bei X" von PSh(C),
  wenn man F := Hom(X, __) nimmt. (Es gilt E tensor_C Hom(X, __) = E(X).)
  Zumindest, wenn C endlich vollständig ist.

  Solche Punkte sind sogar wesentlich.

  Die wesentlichen Punkte von PSh(C) sind genau die winzigen Objekte von
  PSh(C), also die Retrakte von darstellbaren Funktoren.

* Sei Set --> E ein Punkt. Dann ist dieser geometrische Morphismus sogar eine
  Einbettung: Wenn E eine Theorie T klassifiziert und der Punkt durch ein
  Modell M gegeben ist, so klassifiziert dieser Untertopos wohl folgende
  Theorie T_M:

  * T
  * Je ein Konstantensymbol für jedes Element von M
  * Axiome, die regeln, dass sich die Funktionssymbole und Relationssymbole von T
    auf den neuen Konstantensymbolen genauso verhalten wie in M
  * top |-_x bigvee_{a in M} x = \ul{a}
  * \ul{a} = \ul{b} |- bigvee { top | a = b }
  * (Man könnte wohl auch noch Funktionssymbole für jede Funktion M --> M
    hinzunehmen, genauso mit höherer Arität und mit Relationen. Ändert aber
    nichts.)

  Denn der Topos, der T_M klassifiziert, ist ziemlich sicher äquivalent zu Set;
  und das Diagramm

      Set ----> E
        \     ^
         v   /
          E_j

  von geometrischen Morphismen kommutiert. (Der Morphismus Set --> E ist durch
  das T-Modell M und der Morphismus Set --> E_j ist durch das T_M-Modell M
  gegeben.) Wir wissen, dass Set --> E_j eine Äquivalenz ist, da E_j abstrakt
  äquivalent zu Set ist und es nur einen geometrischen Morphismus Set --> Set
  gibt.


=== Vollständigkeit und Kovollständigkeit

* Klar: Das Argument mit Monaden oder CAHA zeigt, dass I-förmige Limiten
  die Existenz von I-förmigen Kolimiten zeigen. (E^op = CAHA(E).)

* Es funktioniert auch umgekehrt: http://mathoverflow.net/a/201749/31233

  The gist of the construction is the following. Let E be an elementary
  topos which has I-indexed coproducts. Then it also has I-indexed
  products. To verify this, let an I-indexed collection (X_i)_i of objects
  in E be given. By the existence of I-indexed coproducts, we can form
  the object (coprod_i X_i) and consider the morphism

      coprod_i X_i ---> coprod_i 1.

  The codomain consists of |I| copies of the terminal object 1.

  In an elementary topos, one can always construct the object of sections
  of a given morphism A --> B, by carving out a suitable subobject of the
  internal Hom(B,A).

  One can now verify that the object of sections of (coprod_i X_i -->
  coprod_i 1) enjoys the universal property of the product of the X_i's.
  (Naively, such a section associates to any element i of I an element of X_i,
  so it's given by an element of the product of the X_i's.)


=== Lokale Topoi

* Ein Topos E --> Set ist genau dann lokal (über Set), wenn gamma_* (globale
  Schnitte) noch einen weiteren Rechtsadjungierten gamma^! besitzt.

  Ein geometrischer Morphismus F --> E ist genau dann lokal, wenn f_* einen
  weiteren Rechtsadjungierten f^! besitzt und dieser sogar E-indiziert ist
  (äquivalent: und f zusammenhängend ist (d.h. dass f^* volltreu ist)).

* Im Fall Sh(X) --> Set schickt gamma^! eine Menge M auf die Garbe

      U |--> Hom({ * | U = X}, M).

* Ist C eine kleine Kategorie mit einem terminalen Objekt, so ist PSh(C)
  ein lokaler Topos.

* Der geometrische Morphismus E/X --> E ist genau dann lokal, wenn X tiny ist.
  (E ein Grothendiecktopos.)

* Lokale geometrische Morphismen besitzen Rechtsinverse.

* Lokale Topoi besitzen stets einen initialen Punkt (nämlich gamma^!).

* Ist F --> E ein lokaler geometrischer Morphismus, wo wird E zu einer
  Kategorie von Garben bezüglich einer Lawvere--Tierney-Topologie in F.
  Und zwar vermöge (f_* -| f^!).

  Diese Situation haben Awodey und Birkedal axiomatisiert. Die
  Abschlussoperationen Sub(X) --> Sub(X), die von der Topologie induziert
  werden, erlauben in diesem Fall Linksadjungierte. nLab schreibt für den
  Linksadjungierten U |-> bU und für den gegebenen Rechtsadjungierten U |-> #U.

  Das Unterobjekt bU kann man als das kleinste #-dichte Unterobjekt von U
  charakterisieren. Es ist auch das Bild der Koeins von (f^* -| f_*), also

      f^* (f_* U) -->> bU ^--> U.

  Im Fall Zar(S) --> Sh(S) ist das (U|_Sh(S))^~ --> bU --> U.

  Achtung: Die Koabschlussoperation U |-> bU ist *nicht* stabil unter Rückzug.
  Gibt also nicht Anlass zu einem internen modalen Operator.

  In F kann man eine volle Unterkategorie der "diskreten Objekte" definieren:
  Ein Objekt X von F heißt genau dann diskret, wenn für alle g : A --> B,
  welche nach Anwenden von f_* (Garbifizieren bezüglich #) zu Isos werden,
  und für alle b : X --> B das Diagramm

       + - - -> A
       |        | g
       ^        v
      X ------> B

  genau auf eine Art und Weise mit einem Morphismus X --> A gefüllt werden
  kann.

  Dann kann man zeigen: Ein Objekt ist genau dann diskret, wenn es im
  wesentlichen Bild von f^* liegt.

  "<==": Hier zeigen wir, dass f^*(X) diskret ist. In diesem Fall setzen wir
  g := T^(f_*(g)^(-1) . b^T).

  "==>": Hier zeigen wir, dass für diskretes X die Eins eps_X : f^*f_*(X) --> X
  ein Iso ist. Da f_*(eps_X) ein Iso ist (mit Inversem eta_(f_*X), hier geht
  die Volltreuheit ein), gibt es nach Voraussetzung einen Morphismus
  psi : X --> f^*f_*(X) mit eps_X . psi = id.

  Dann rechnet man: (psi . eps_X)^T = f_*(psi) = f_*(eps_X)^(-1) = eta_(f_*X) =
  id^T, also psi . eps_X = id.

  Fazit: f^* : E --> F ist äquivalent zur Inklusion der diskreten Objekte.
  Die Äquivalenz ist durch f^* gegeben.

  Wenn man in der internen Sprache von F die Aussage formuliert, dass ein
  Objekt X diskret ist, bedeutet das wohl, dass X in allen F/T diskret ist.
  Mir ist nicht ganz klar, ob das gleichbedeutend damit ist, dass X schon in F
  diskret ist. Es könnte stimmen. Es könnte aber auch nicht stimmen, denn man
  muss hier vorsichtig sein: Sind die entsprechenden Funktoren auf richtige Art
  und Weise indiziert?

* Lokale geometrische Morphismen bewahren Objekte von dedekindschen reellen
  Zahlen.

* Sei f : X --> Y eine stetige surjektive Abbildung von nüchternen
  Räumen, sodass jede Faser f^{-1}[y] einen Brennpunkt c_y besitzt
  und sodass y |--> c_y als Abbildung Y --> X stetig ist. Dann ist
  f : Sh(X) --> Sh(Y) ein lokaler geometrischer Morphismus.


=== Surjektive geometrische Morphismen

* Ein geometrischer Morphismus f : F --> E heißt genau dann surjektiv, wenn
  f^* treu ist (äquivalent: konservativ ist).

* Set/X --> Set ist genau dann surjektiv, wenn X bewohnt ist.

  Die Rückrichtung sieht man so: Zunächst sieht man, dass der Rückzug zwei
  Abbildungen f und g genau dann miteinander identifiziert, wenn (X bewohnt ==> f = g).
  Der Morphismus ist also genau dann surjektiv, wenn aus (X bewohnt ==> f = g)
  schon f = g folgt. Fertig mit f, g : 1 --> Omega, f(*) = top, g(*) = [| X bewohnt |].

* Als Korollar ist E/X --> E genau dann surjektiv, wenn X bewohnt ist,
  denn in meiner Masterarbeit habe ich ja gezeigt, dass interne Surjektivität
  äquivalent zu externer Surjektivität ist.

* Es gibt eine Charaktieriserung der Örtlichkeiten, die surjektiv über dem
  Punkt liegen: Es sind genau die, für die es einen Situs gibt, in dem jede

=== Begriffe von Injektivität

Roswitha Harting untersucht in
http://www.tandfonline.com/doi/pdf/10.1080/00927878308822853 diverse
Injektivitätsbegriffe in einem Topos E. Unter anderem folgende:

  (II)_1 <==> (II)_2: Der Funktor [__, X] : E^op --> E schickt
  Monomorphismen auf Epimorphismen.

  (LI)_1 <==> (LI)_2: Der Funktor [__, X] schickt Monomorphismen auf solche
  Morphismen, die für jedes globale Element des Zielobjekts lokal Urbilder
  zulassen.

  (III): Alle p^*(X) haben die Eigenschaft (II).

  (ILI): Alle p^*(X) haben die Eigenschaft (LI).

Wir können auch noch ergänzen:

  (SI): Aus Sicht der internen Sprache ist X injektiv.

All diese Eigenschaften sind äquivalent.

  (II) =====> (LI) =====> (ILI) =====> (III) =====> (II)
        triv                             ^^   triv
                                     triv||
                                         vv
                                        (SI)

Der Beweis von (LI) ==> (ILI) nutzt den Linksadjungierten p_!. Er geht etwa so:

    Sei ein Mono S --> T in E/A gegeben. Sei ein Morphismus S --> p^*(X) in E/A
    gegeben. Da X Eigenschaft (LI) hat, hat der Morphismus p_!(S) --> p_!(T)
    bezüglich p_!(S) --> p_!(p^*(X)) --> X lokal einen Kolift. Es gibt also
    einen Epi q : B -->> 1 und einen Morphismus g : q^*(p_!(T)) --> q^*(X)
    mit g . (q^*(p_!(S)) --> q^*(p_!(T))) = (q^*(p_!(S)) --> q*(X)).
    Den Morphismus g kann man längs p' zurückziehen. Dann noch T --> p^*(p_!(T))
    einschieben, fertig.

    Dabei sind p' und q' die durch Rückzug entstehenden Morphismen aus p bzw. q
    (die dann in A x B starten).

Das kann man auch noch einfacher formulieren:

    Wir wissen, dass p_* . [__, p^*(X)]_{E/A} = [__, X]_E . p_!.
    Der Funktor p_! bewahrt Monomorphismen. Also ist nur noch zu zeigen,
    dass p_* "Pseudo-Epis" (solche, die für globale Elemente lokal Urbilder
    besitzen) reflektiert. Das geht so:

    Sei f : X --> Y ein Morphismus in E/A derart, dass p_*(f) in E ein
    Pseudo-Epi ist. Sei y : 1 --> Y ein globales Element in E/A.
    Das entspricht einem globalen Element y von p_*(Y). Somit existiert
    ein Epi q : B -->> 1 in E und ein Element x : B --> p_*(X) sodass
    p_*(f) . x = y . q. Dann ist T^x : p^*(B) --> X das gesuchte lokale Urbild,
    denn f . T^x = q' . y.

Der Beweis von (ILI) ==> (III) ist wieder recht einfach (wie bei der
Vereinfachungsregel für die interne Sprache im Fall "forall forall").

Das ganze funktioniert gleichermaßen für reine Objekte und für Objekte
abelscher Gruppen.

Nun kann man in Sh(X) auch von Injektiven bezüglich des externen Hom-Funktors
sprechen. (Allgemeiner geht das für jeden Set-Topos E. Und natürlich kann man
auch relativieren.) Da gibt es folgende Aussagen:

* Eine Garbe I ist genau dann (extern-)injektiv, wenn sie auf einer Überdeckung
  (extern-)injektiv ist. Dabei ist die Richtung "==>" trivial (nutze
  Extra-Linksadjungierten zur Einschränkung). Die Richtung "<==" geht so:

      Zunächst ist eine Garbe genau dann welk (Einschränkungsabbildungen alle
      surjektiv), wenn sie auf einer Überdeckung welk ist. (Das zeigt man mit
      Zorn.) Außerdem bewahrt der globale-Schnitte-Funktor Gamma kurze exakte
      Sequenzen, wenn die vorderste Garbe welk ist.

      Für (extern-)injektive Garben J ist HOM(??, J) welk (siehe garben.txt).
      Also ist HOM(??, I) welk (nach Voraussetzung ist sie es lokal).

      Der Funktor HOM(__, I) bewahrt kurze exakte Sequenzen. Dafür würde schon
      genügen (das ist nach Harting tatsächlich äquivalent dazu), dass I nur
      aus Sicht der internen Sprache injektiv ist.

      Somit bewahrt Hom(__, I) = Gamma . HOM(__, I) kurze exakte Sequenzen.

* Schon die triviale Richtung zeigt, dass injektive Garben aus interner Sicht
  injektiv sind.

* Ich glaube, dass auch die Umkehrung gilt -- mit demselben Argument:
  Denn ich glaube, dass auch für Garben I, die nur aus lokaler Sicht injektiv
  sind, Garben der Form HOM(??, I) welk sind. Ferner ist HOM(__, I) exakt,
  da I aus interner Sicht injektiv ist. Somit bewahrt Hom(__, I) = Gamma . HOM(__, I)
  kurze exakte Sequenzen.

  Das war jetzt für den Fall von Garben abelscher Gruppen. Es stimmt aber auch
  für Garben von Mengen. Das geht zum Beispiel von Hand:

      Sei I aus interner Sicht injektiv. Wir wollen zeigen, dass I
      (extern-)injektiv ist. Sei dazu ein Monomorphismus iota : E --> F
      und ein beliebiger Morphismus f : E --> I gegeben.

      Mit dem Zornschen Lemma finden wir einen maximalen Kolift f' : F|_U --> I|_U.

      Wir betrachten jetzt das Injektivitätsdiagramm

          E cup j_!(F|_U) -----> F
               |
               | (gegeben durch f bzw. f')
               v
               I.

      Die Vereinigung findet dabei als Unterobjekte von F statt (E ist ein
      solches vermöge iota, j_!(F|_U) ist es vermöge der Koeins). Der
      Morphismus nach I hinein ist wohldefiniert, da f und f' auf E|_U
      übereinstimmen. Nach Voraussetzung gibt es jetzt für einen beliebigen
      Punkt x von X eine offene Umgebung V und einen Kolift f'' : F|_V --> I|_V
      dieses Diagramms. Dieser Kolift stimmt auf U cap V mit f' überein.
      Außerdem ist er auch ein Kolift von E --> I.

      Somit können wir f' und f'' zu einem größeren Kolift (auf U cup V
      definiert) verkleben. Wegen der Maximalität von f' ist U cup V <= U,
      also V <= U.

      Das zeigt U = X.

  Cooler so: Sei I wieder aus interner Sicht injektiv, iota : E --> F
  ein Mono und f : E --> I. Wir betrachten die durch die interne Formel
  definierte Garbe

      G := { f' : [F,I] | f' . iota = f }.

  Wir möchten zeigen, dass diese Garbe einen globalen Schnitt besitzt.
  Dazu müssen wir nur ihre Welkheit nachweisen. Aber das können wir intern:

      Sei K <= G ein Subsingleton. Wir betrachten das Injektivitätsdiagramm

         iota[E] cup F' --> F   mit F' = { s : F | K bewohnt }
                |
                |
                v
                I,

      wobei die vertikale Abbildung ein s aus F' auf f'(s) schickt, wobei f'
      irgendeine Abbildung in K ist, und ein iota(e) auf f(e) schickt (das ist
      wohldefiniert). Somit existiert ein Kolift f' : [F,I]. Dieser liegt schon
      in G. Außerdem liegt er in K, wenn K bewohnt wäre.

  Der gleiche Beweis kann auch im linearen Fall verwendet werden; betrachte
  dann iota[E] + F' mit F' = { s : F | s = 0 oder K bewohnt }.

=== Generische Elemente

Sei X ein Objekt eines Topos E. Wir möchten einen neuen Topos E' --> E,
in dem es ein generisches Element (des Abbilds von) X gibt.

Das geht so: E' := E/X. Das Abbild von X ist dann X x X.
(Abbild-nehmen ist X x __.) Das gesuchte generische Element ist dann
X --> X x X über X.

Der geometrische Morphismus ist

       f_* : E/X -----> E    (abhängiges Produkt nehmen)
    f^{-1} :   E -----> E/X  (mit X multiplizieren, also konstante Familie nehmen)

* Im Übrigen ist E/X --> E eine geometrische Einbettung, falls X ein
  Subsingleton ist. In diesem Fall ist f_*(E --> X) = [X, E] (gewöhnliches
  inneres Hom). Die zugehörige Lawvere--Tierney-Topologie ist (X bewohnt ==> __).

  Und E/X --> E ist auch nur dann eine Einbettung, falls X ein Subsingleton
  ist: Prüfe das am besten mit dem Kriterium, dass die Koeins

      X x [X, A]_{nur Schnitte} ---> A   (Morphismus in E/X)

  für alle Objekte A aus E/X ein Iso ist. Speziell für A = (X x X --> X)
  wird die nämlich zu

      X x [X, X] ---> X x X, (x,f) |--> f(x)

  und zeigt, wenn sie ein Iso ist, dass X ein Subsingleton ist: Seien x, x' : X.
  Dann sind die Bilder von (x, id_X) und (x, konstant_x) gleich. Also id_X =
  konstant_x. Einsetzen von x' zeigt: x' = x.

  Übrigens sind die offenen Untertopoi von E genau diejenigen von der Form
  E/X --> E mit E einem Subsingleton. Das ist mehr oder weniger unmittelbar
  klar, denn ein offener Untertopos ist von der Form E_Box mit Box = (U ==> __),
  wobei U ein Subsingleton ist. So ein Topos ist aber gleich E/U.

* Im Fall, dass X ein Subsingleton ist, gilt:

      Hom_{Top^op}(E/X, F) =
          Hom_Top(F, E/X) =
          Hom_Top(F, E)_{mit Bild Garben} =
          Hom_Top(F, E)_{mit Urbild von A bewohnt} =
          Hom_{Top^op}(E, F)_{wobei A auf 1 geschickt wird}

=== Automatische Kovollständigkeit

http://math.stackexchange.com/questions/1474142/finite-limits-exponentiation-and-sub-object-classifiers-imply-finite-co-limits

The idea of Paré's proof, specialized to the case of the topos Set, is the
following. First, show that Set^op is equivalent to the category of complete
atomic Heyting algebras (using the contravariant power set functor). (In
classical logic, you can also say "complete atomic Boolean algebras".)

Then observe that for any index category I, if Set possesses all I-shaped
limits, then so does the category of complete atomic Heyting algebras.
Transporting this observation along the equivalence of categories, we see that
Set possesses all I^op-shaped colimits.

For the general case of an arbitrary topos E, one can mimic these arguments
using diagrammatic instead of element-based notation. One could also employ the
internal language (if one makes sure that the reasoning in justifying its use
is not circular).


=== Analogie zu Relativitätstheorie

Mike in
https://golem.ph.utexas.edu/category/2011/06/flat_functors_and_morphisms_of.html#c038686:

Similarly, to say that the internal logic of a topos is constructive, but that
this fact is just another piece of mathematics formulated in set theory and
good old classical logic, misses the point. Even the conclusion that because of
this, it is better to formulate mathematical arguments constructively so that
they can be applied inside any topos, is not the whole story. The point is that
any topos is really as good as any other, including what we like to think of as
the “good old” topos of sets that satisfy classical logic and (maybe) the axiom
of choice.

In fact, there is also a nice anology to be drawn between LEM+AC and the
property of being an inertial reference frame, so that the inclusion of
non-classical toposes corresponds to the theory of general relativity. (And, of
course, it’s worth noting that the reference frame we live in is not inertial!)

=== Generische Primzahl

Der Topos, der eine generische natürliche Zahl enthält, ist einfach
Set/N oder äquivalent Sh(N). (Das Objekt natürlicher Zahlen in Sh(N) ist
die Garbe der N-wertigen Funktionen auf N und die generische natürliche
Zahl in Sh(N) ist ein bestimmter globaler Schnitt dieser Garbe, nämlich
die Identitätsfunktion.)

Da die Bedingung an eine natürliche Zahl, Primzahl zu sein, eine
geometrische Sequenz ist, ist der Untertopos, wo die generische
natürliche Zahl prim ist, einfach Set/P bzw. Sh(P). Dabei ist P
die Menge der Primzahlen.

(Als abstrakte Topoi sind Set/P und Set/N äquivalent, aber die
kanonische geometrische Einbettung Set/P --> Set/N ist keine
Äquivalenz.)

Heiße wie beim Mittagessen die generische Primzahl in Set/P "p0".

Dann gibt es, für eine gewöhnliche Primzahl p, keinen Ringhomo
(Set, F_p) --> (Set/P, F_p0): So ein Morphismus wäre topologisch durch
den eindeutigen Morphismus Set/P --> Set gegeben und ringtheoretisch
durch F_p --> F_p0 (in Set/P); so einen Morphismus gibt es aber nicht,
da in Set/P nicht die Aussage stimmt, dass p ein Vielfaches von p0 ist.

Es gibt aber, für jede gewöhnliche Primzahl p, einen Ringhomo
(Set/P, F_p0) --> (Set, F_p): topologisch ist dieser durch Halmbildung
bei p gegeben und ringtheoretisch durch die Identität (denn der Keim der
generischen Primzahl bei p ist einfach p).

Unser Körper F_p0 besitzt also durchaus Morphismen in alle F_p; aber
leider nicht nach Z.


=== Abhängiges Auswahlaxiom

Natürlich gilt nicht in jedem Topos das abhängige Auswahlaxiom. Allerdings
schon, wenn man den Übergang zu einer offenen Überlagerung gestattet.

http://www.ams.org/journals/tran/1986-295-02/S0002-9947-1986-0833712-3/S0002-9947-1986-0833712-3.pdf
Seite 5

Und zwar sollte das so gehen: Sei X die Menge, aus der wir auswählen wollen.
Sei T ein "Baum mit unendlichen Zweigen": eine Menge von endlichen Folgen von
Elementen aus X mit folgenden Eigenschaften:

    (1) <> in T  (leere Folge)
    (2) u <= v und v in T  ==>  u in T  (Präfixe liegen wieder in T)
    (3) u in T  ==>  exists x:X. (u,x) in T

Dann sollte man die Örtlichkeit der Zweige von T betrachten können:

    Propositionen: "f(n) = x" für x:X und n >= 0

    Axiome:
    * top |-_n bigvee_{x:X} (f(n) = x)  (für alle n >= 0)
    * f(n) = x wedge f(n) = x' |-_n bigvee { top | x = x' }  (für alle x, x' : X und n >= 0)
    * f(0) = x0 wedge ... f(n) = xn |-_n bigvee { top | <x0,...,xn> in T }
          (für alle n >= 0 und x0, ..., xn : X)

Der Rahmen der Offenen dieser Örtlichkeit sollte einfach gegeben sein durch
den Rahmen derjenigen nach-oben-abgeschlossenen Teilmengen S von T mit:

    forall <x1,...,xn>.
        [forall x. <x1,...,xn,x> in S] ==> <x1,...,xn> in S

Infimum durch Schneiden, Supremum durch Vereinigen und Abschließen:

    S |--> j(S) = { xs | forall x. <xs,x> in S }

=== Flache Funktoren

* Vor flachen Funktoren muss man echt keine Angst haben.

* Geometrische Morphismen E --> PSh(C) stehen in kanonischer
  Eins-zu-Eins-Korrespondenz zu Modellen der folgenden geometrischen Theorie.

  Sorten: je eine, F(X), für jedes Objekt X aus C

  Funktionssymbole: je eins, F(f), für jeden Morphismus f in C

  Axiome:

  1. Funktoraxiome
  2. Flachheit:
     * top |- bigvee_X (F(X) bewohnt)
     * top |-{u : F(X), v : F(Y)} bigvee_{Z, f : Z --> X, g : Z --> Y}
                    exists w : F(Z).
                        F(f)w = u  wedge  F(g)w = v
     * F(f)u = F(g)u |-_{u : F(X)} bigvee_{Z, h : Z --> X, f . h = g . h}
                    exists w : F(Z).
                        F(h)w = u
  3. Überdeckungsbewahrung:
     * top |-_{u : F(X)} bigvee_i exists v : F(X_i). F(X_i --> X)v = u
          für jede Überdeckung (X_i --> X)_i

  Siehe etwa http://www.tac.mta.ca/tac/volumes/12/5/12-05.pdf.

* Flache Funktoren bewahren endliche Limiten. Kann man schnell einsehen.
  Die Voraussetzung der Freiheit (letztes Flachheitsaxiom) muss man mehrere
  Male einsetzen.

* In manchen Fällen sind flache Funktoren durch weniger Daten gegeben,
  als es auf den ersten Blick aussieht. Etwa sind flache Funktoren F auf
  Ring_fp^op einfach durch F(Z[X]) (zusammen mit der Ringstruktur auf dieser
  Menge) gegeben. Das liegt daran, dass jedes Objekt aus Ring_fp^op endlicher
  Limes von Z[X]'en ist und dass sich Morphismen nur aus polynomiellen
  Ausdrücken zusammensetzen.
