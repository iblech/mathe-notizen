=== Coquands Spektrum

Sei A ein Ring.

Definiere L_A := Rad_fin(A), ist ein distributiver Verband (kein Rahmen).
Hat die Basis L_A' := { sqrt(f) | f \in A }.

Habe die auf L_A' definierte Garbe

    J |--> A[S_J^(-1)], wobei S_J = { c \in A | J \subseteq sqrt(c) }.
    (Bemerkung: Falls J = sqrt(f), ist Invertierung an S_J dasselbe wie
    Invertierung von f.)

Kann diese Garbe auf ganz L_A fortsetzen. Auf diese Weise wird L_A zu einem
geringten distributiven Verband.

Über Lokalitätsbegriffe spricht Coquand erstmal nicht.


=== Rahmen der Radikalideale

* Rad(A), die Menge aller Radikalideale, ist ein Rahmen.

* Elemente der Form sqrt((x_1,...,x_n)) sind kompakt.

* sqrt(a cap b) = sqrt(a) cap sqrt(b) = sqrt(a * b).

* (a --> b) = (b : a) = { x | xa <= b }.
  (Für beliebige Ideale, nicht unbedingt Radikalideale, gilt:
  c <= (sqrt(a) : b)  <==>  c cap b <= sqrt(a).

* Für ein Radikalideal a ist die Menge

      Ja := { x | a \subseteq sqrt(x) }

  eine saturierte multiplikativ abgeschlossene Teilmenge. Für a = sqrt(f)
  ist Lokalisierung von Ja dasselbe wie Invertierung von f.

  Auf Rad(A) kann man kanonisch die Prägarbe

      O(a) := A[(Ja)^(-1)]

  definieren. Für endliche Überdeckungen erfüllt die vermutlich das
  Verklebeaxiom, für unendliche aber vermutlich nicht.
  Update: Für standardoffene Überdeckungen (auch unendliche) ist das
  Verklebeaxiom erfüllt. Aber dann nicht für beliebige. Man sollte man explizit
  ein Gegenbeispiel mit A^2 \ {0} vorführen!

  Auf jeden Fall kann man intern in Sh(Rad(A)) einfach die konstante Garbe A
  an J lokalisieren. Das gibt die Strukturgarbe.

* Kurze Memo: Ein Radikalideal ist *nicht* genau dann endlich erzeugt,
  wenn es von der Form sqrt((x_1,...,x_n)) ist. Ein Beispiel ist das
  Radikalideal sqrt((0)) im Ring Z[X_1,...]/(X_1^1,X_2^2,...). Das ist ziemlich
  sicher gleich (X_1,...) und nicht endlich erzeugt.

* Es gibt auch den distributiven Verband der Radikalideale der Form
  sqrt((x_1,...,x_n)). Dessen Vervollständigung zu einem Rahmen ist Rad(A).

  Ist phi : Rad(A)_fg --> L ein Morphismus in einen Rahmen L, so können wir
  eine Fortsetzung phi' : Rad(A) --> L durch

      phi'(a) := bigvee_{b <= a, b in Rad(A)_fg} phi(b)

  definieren. Um Verträglichkeit von phi' mit bigvee einzusehen, beobachte,
  dass für b in Rad(A)_fg mit b <= bigvee a_i gilt: Es gibt endlich viele u_j
  mit u_j in a_j und b = bigvee_j (b wedge sqrt((u_j))), also phi(b) =
  bigvee_j phi(b wedge sqrt((u_j))); dann fertig mit der Beobachtung, dass
  b wedge sqrt((u_j)) endlich erzeugt und Teilmenge von a_j ist.

* Die örtlichkeitstheoretischen Punkte von Rad(A) stehen in
  Eins-zu-Eins-Korrespondenz zu den Filtern in A. Das hat einen tieferen Grund,
  denn Rad(A) ist die Örtlichkeit zur geometrischen Theorie eines Filters in A;
  Punkte sind daher Modelle in Set, d.h. tatsächliche Filter in A.

  Explizit sieht die Korrespondenz wie folgt aus:

  Sei F ein Filter in A. Dann ist M := { I | I cap F bewohnt } ein vollständig
  primer Filter in Rad(A), d.h. ein Punkt von Rad(A).

  Sei umgekehrt M ein vollständig primer Filter in Rad(A). Dann ist
  F := { x | sqrt(x) in M } ein Filter in A.

  In Primidealsprache gibt es also folgende Korrespondenz:

                Primideale von A  ~~~  Punkte von Rad(A)
                               p  |->  { I | I cap p^c bewohnt }
        { x | sqrt(x) \not\in M } <-|  M

* Spec(A) ist lokal kompakt. Sei dazu ein Radikalideal J gegeben.
  Wir zeigen: J <= bigvee { K | K << J }. Das ist leicht, denn ist f aus J,
  so ist f Element von sqrt(f), und das ist << J.

  Denn: Sei S gerichtet mit J <= bigvee S. Dann sqrt(f) <= bigvee S.
  Also existiert ein Ideal L aus S mit sqrt(f) <= L.

* Spec^pcoh(A) ist vermutlich nicht lokal kompakt.


=== Verklebeaxiom in einem Beispiel

Sei A ein Ring. Sei 1 = sum_i z_i x_i eine Zerlegung der Eins. Seien Brüche

    s_i = a_i/b_i in A[S_i^(-1)]

gegeben, wobei S_i = { c | x_i in sqrt(c) }. Also gibt es (oBdA eine)
natürliche Zahl n und Ringelemente u_i mit

    x_i^n = u_i b_i.

Gelte s_i = s_j in den zweifach lokalisierten Ringen. Das bedeutet (oBdA), dass
es eine natürliche Zahl m und Ringelemente c_ij, v_ij mit

    c_ij b_i a_j = c_ij b_j a_i,
    (x_i x_j)^m = v_ij c_ij

gibt. Setze dann s := sum_i #_i x_i^m a_i u_i, wobei die #_i so gewählt sind,
dass 1 = sum_i #_i x_i^(m+n) gilt. Dann gilt s/1 = a_j/b_j in A[S_j^(-1)], denn

    x_j^m b_j s = ... = x_j^m a_j.


=== Duale Algebra der Radikalideale ist kein Rahmen

Beispiel in Z: Das Radikalideal

    Schnitt über alle sqrt(p), p >= 3
    addiert zu (2)

ist gleich (2) und daher echt kleiner als

    Schnitt über alle (sqrt(p) addiert mit (2)), p >= 3,

was nämlich gleich (1) ist.


=== Spektrum als Lokalisierung an universellem Filter
(sollte lieber "generisch" statt "universell" sagen!)

Def.: Ein Filter F eines Rings A ist eine Teilmenge mit
1) 1 in F
2) xy in F  <==>  x, y in F
3) nicht: 0 in F
4) x + y in F  ==>  x in F oder y in F

Prop.: A ist ein lokaler Ring <==> A^x ist ein Filter.

Prop.: Sei S saturierte mult. abg. Teilmenge. Dann:
    S Filter <==> A[S^(-1)] lokal.

Beob.: In X = Spec A habe gamma^*(A). Das ist ein Ring, jedoch nicht unbedingt
ein lokaler. Ferner habe den universellen Filter F auf gamma^*(A), auf offenen
Teilmengen definiert durch

    F(U) = { f : U --> A stetig | f(p) \not\in p f.a. p \in U }.

Die Lokalisierung von gamma^*(A) nach diesem Filter liefert gerade O_X.

Es gibt auch das universelle Primideal Z, mit

    Z(U) = { f : U --> A stetig | f(p) \in p f.a. p \in U }.

Es gilt in der internen Sprache:

    forall f : gamma^*(A). nicht (f in F) <==> f in Z.

* D(f) |== g in F  <==>  f in sqrt((g))
* D(f) |== g in P  <==>  fg nilpotent

* Ferner: Ist M ein A-Modul, so ist gamma^*(M)[F^(-1)] gerade M~
  (die Tilde-Konstruktion). Denn:

  M[F^(-1)]_pre (dabei M konstante Prägarbe) bildet D(f) auf
  M[Sat_f^(-1)] ab. Man weiß, dass auf den standardoffenen Teilmengen diese
  Zuordnung die Garbenaxiome erfüllt. Also gilt:

      gamma^*(M)[F^(-1)](D(f)) =
      a(M[F^(-1)]_pre)(D(f)) =
      M[F^(-1)]_pre(D(f)) =
      M[f^(-1)].

  Dabei nutze aus, dass die Garbifizierung einer Prägarbe, welche bzgl. einer
  Basis B eine Garbe ist, eingeschränkt auf B die ursprüngliche Prägarbe ist.
  Nutze außerdem aus, dass a(colim_pre E_i) = a(colim_pre a(E_i)).

* Spec A |== exists f : gamma^*(A). F = { g : gamma^*(A) | f in sqrt((g)) }
  ist völlig falsch.

* Spec A |== forall x : gamma^*(A). x in F  v  x in Z
  ist falsch.

  Betrachte A = Z, x = 2. Wenn das stimmen würde, gäbe es eine Zerlegung der
  Eins 1 = sum_i f_i mit f_i in sqrt((2)) bzw. f_i * 2 nilpotent.
  Mindestens eines der f_i ist ungerade. Für dieses kann keine der beiden
  Alternativen gelten.

* Spec A |== neg (x in F  wedge  forall y. xy - 1 in F)
  ist für x in A vermutlich auch falsch. P ist also kein maximales Ideal, bzw.
  F kein minimaler Filter.

* Spec A |== forall y. xy - 1 in F
  ist äquivalent zu xy - 1 invertierbar für alle y,
  also zu x in Jac, dem Jacobson-Radikal.

  Verfeinert: D(f) |== forall y. xy - 1 in F
  genau dann, wenn f in sqrt(xy - 1) für alle y.

* Spec A |== x in F  vee  forall y. xy - 1 in F
  ist falsch. Gegenbeispiel: A = Z, x = 2. In einer Zerlegung der Eins muss
  mindestens eines der f_i ungerade sein. Für dieses kann weder f_i in sqrt(2)
  noch f_i in sqrt(3), sqrt(5), sqrt(7), ... gelten.

* D(f) |== F cap I bewohnt  ==>  D(f) |== f^N in I für ein N >= 0,
  falls:

  1. (U |== a in I) ==> (U bewohnt ==> Spec A |== a in I) für alle
     U <= D(f) offen, a in A.

     Hinreichend dafür: I = { x : AA | phi(x) }, wobei in phi nur konstante
     Garben vorkommen.

  2. D(f) |== I Ideal.

* Der Filter F ist zwar "generisch". Das heißt aber nicht, dass jeder
  Filter von AA in Sh(Spec A) eine Obermenge von F wäre. "Man muss noch
  zurückziehen."

  Konkretes Gegenbeispiel: Sei A lokal. Dann umfasst der Filter AA^x der
  invertierbaren Elemente nicht den generischen Filter. Denn

      D(f) |== f in F,

  aber

      D(f) |== f in AA^x

  bedeutet: Ist D(f) bewohnt, so ist f in A invertierbar; also: f ist nilpotent
  oder invertierbar. Das ist im Allgemeinen falsch (zum Beispiel A = Z_(2), f = 2).


=== Initialer lokaler Ring

Einen initialen lokalen Ring kann es nicht geben. Denn sei A dieser Ring. Wähle
eine Primzahl p. Da es einen lokalen Ringhomo A --> Q gibt, ist p in A
invertierbar. Andererseits gibt es einen (lokalen, aber egal) Ringhomo A --> F_p.
Also ist p in F_p invertierbar. Das stimmt nicht.

Wenn man bereit ist, den initialen lokalen Ring in beliebigen Topoi zu suchen,
dann gibt es ihn natürlich schon: Er ist die Strukturgarbe von Spec(Z).


=== Spektrum als Lindenbaum-Algebra einer propositionalen geometrischen Theorie

Atomare Propositionen sind: "x in F" für x in A.

Axiome (eigentlich Axiomschemata):

1. top        |- 1 in F.
2. ab in F  -||- a in F und b in F.
3. a+b in F   |- a in F oder b in F.
4. 0 in F     |- bot.

Gibt es eine Möglichkeit, Minimalität des Filters F als geometrische
Implikation zu formulierenn? Ein Vorschlag:

    x in F und bigwedge_u (ux-1 in F) |- bot.

Wenn die Konjunktion endlich ist, ist das eine geometrische Implikation.

Komische Situation: Ich weiß ja, dass MaxSpec A ein Teilraum von Spec A ist.
Also muss es auch einen Kern auf dem Rahmen von Spec A geben, sodass die
zugehörige Unterörtlichkeit MaxSpec A ist.


=== MaxSpec (das Spektrum bestehend aus maximalen Idealen)

* MaxSpec(A) ist im Allgemeinen nicht nüchtern. Man denke etwa in
  MaxSpec(Z) an den vollständig primen Filter { U | U bewohnt },
  der nicht von der Form { U | m in U } für ein maximales Ideal m ist.

* Der Raum der fast maximalen Primideale, AlmostMaxSpec(A), ist schon
  nüchtern. Er ist der Raum der Punkte von Jad(A) (siehe unten).

* Die Nüchternifizierung von MaxSpec(A) ist AlmostMaxSpec(A).
  Denn wenn man den Kern für MaxSpec(A) herleitet, erhält man den von
  AlmostMaxSpec(A).

* Ein Ideal I heißt genau dann "fast maximal", wenn es Schnitt
  von maximalen Idealen ist. Das ist genau dann der Fall, wenn in A/I das
  jacobsonsche Radikal Null ist. Oder äquivalent:

      (forall h. exists u. (1 - xh)u - 1 in I) ==> x in I.

* Die Zuordnung I |-> I^ mit

      I^ := { x | forall h. exists u. (1 - xh)u - 1 in I }
          = { x | forall h. 1 - xh invertierbar in A/I }
          = { x | x liegt im jacobsonschen Radikal von A/I }
          = bigcap_{m maximal, I <= m} I

  ist ein Kern auf Rad(A). Die entstehende Unterörtlichkeit nenne ich Jad(A).

  (Zum Vergleich: sqrt(I) = bigcap_{p prim, I <= p} I.)

  [ Manueller Nachweis, direkt anhand der konstruktiv sinnvollen Definition,
  dass I |-> I^ ein Kern ist: I <= I^ ist klar. Sei x in I^^. Um x in I^
  nachzuweisen, sei h beliebig. Dann gibt es ein u mit (1 - xh)u - 1 in I^.
  Somit existiert ein v mit (1 - ((1 - xh)u - 1) (-1))v - 1 in I. Der lange
  Ausdruck vereinfacht sich zu (1 - xh)uv - 1 in I. Das war zu zeigen.
  Um I^ cap J^ <= (I cap J)^ nachzuweisen, sei x aus I^ cap J^ und sei h
  beliebig. Dann gibt es u und v mit (1 - xh)u - 1 in I und (1 - xh)v - 1 in J.
  Somit (1 - xh) (u - (1-xh)uv + v) - 1 = ((1 - xh)u - 1) * (1 + (1-xh)v) =
  ((1 - xh)v - 1) * (1 + (1-xh)u) in I cap J. Um zu zeigen, dass I^ überhaupt
  abgeschlossen unter Addition ist, betrachte (1-(x+y)h)uv-1, wobei (1-xh)u-1
  in I und (1-yuh)v-1 in I. ]

  Aus Johnstone (Almost maximal ideals) kann man herauslesen:

      I^ = { x | forall h. (x,h) = (1) ==> exists u. (h,u) = (1) }

  Speziell:

      jac = (0)^ = { x | forall h. (x,h) = (1) ==> h inv. }.

* Der Raum der Punkte von Jad(A) ist der Raum der fast maximalen Primideale.

  Sei F ein vollständiger primer Filter in Jad(A).
  Dann ist p := { x | (x)^ nicht in F } ein fast maximales Primideal.
  Die Fast-Maximalität zeigt man so: Sei x in p^. Angenommen x nicht in p.
  Dann (x)^ in F. Somit:

      (x)^ = (x)^ cap p^ = ((x) * p)^ = bigvee_{(a)^ nicht in F} (ax)^ in F.

  Das bigvee ist in Jad. Da F vollständig prim ist, gibt es ein (a)^ nicht in F
  mit (ax)^ in F. Das kann nicht sein.

  Sei p ein fast maximales Primideal.
  Dann ist F := { J | J cap p^c bewohnt } ein vollständig primer Filter.

* Kann man Fast-Maximalität als geometrische Implikation formulieren?
  "Ja." Man muss für jedes x in A und jede Funktion u : A --> A
  folgendes Axiom hinzufügen:

      x in p^c |- bigvee_{h in A} ((1 - hx) u(h) - 1) in p^c.

  Dann kann man klassisch (!) zeigen, dass die Örtlichkeit der entstehenden
  Theorie gleich Jad(A) ist. Aber konstruktiv geht es nicht.

* In MaxSpec(A) und AlmostMaxSpec(A) gilt: D(f) <= D(g) genau dann, wenn
  f in (g)^. Denn:

         D(f) <= D(g)
    <==> forall m. f nicht in m ==> g nicht in m
    <==> forall m. g in m ==> f in m
    <==> f in bigcap_{m, g in m} m
    <==> f in (g)^.

  Es spielt keine Rolle, ob die Quantifikation über alle maximalen Ideale geht,
  über alle fast maximalen Primideale oder sogar über alle fast maximalen
  Ideale.

* AlmostMaxSpec(A) ist kompakt. (Denn I^ = (1) nur dann, wenn schon I = (1).)

=== Wann ist das Spektrum einpunktig?

Sei A ein lokaler Ring.

* Genau dann erfüllt der lokal geringte Raum (pt, A) die universelle
  Eigenschaft von Spec A (nämlich, dass Morphismen aus beliebig lokal geringten
  Räumen hinein durch Ringhomos in die andere Richtung auf globalen Schnitten
  gegeben sind), wenn jeder Ringhomo A --> B in einen lokalen Ring schon lokal ist.

  Dabei habe ich evtl. ein kleines bisschen klassische Logik verwendet, durch
  Betrachtung von Halmen. Neu machen mit Örtlichkeiten!

* In klassischer Logik ist das genau dann der Fall, wenn A genau ein Primideal
  besitzt:

  * Sei p irgendein Primideal. Betrachte den Homo A --> A_p.
    Sei x ein Element aus A, das nicht invertierbar ist (also im maximalen
    Ideal liegt). Dann ist nach Voraussetzung auch das Bild in A_p nicht
    invertierbar, also liegt x in p.
    Somit m \subseteq p. Die umgekehrte Richtung gilt sowieso.

  * Wenn A nur ein Primideal besitzt, ist das maximale Ideal als Schnitt aller
    Primideale das Nilradikal. Also ist jedes Element entweder invertierbar
    oder nilpotent. Damit folgt sofort die Behauptung.

* Konstruktiv hinreichend dafür sind etwa folgende Bedingungen:

  * nilpotent oder invertierbar
  * nicht nilpotent ==> invertierbar  [S^(-1) O_X erfüllt das zum Beispiel,
    wenn X reduziertes Schema]

  Es genügt aber scheinbar nicht:

  * nicht invertierbar ==> nilpotent

* Wenn man Spec A als Rahmen der Radikalideale von A (nicht nur der endlich
  erzeugten) realisiert, so ist Spec A genau dann einpunktig, wenn jedes
  Element von A nilpotent oder invertierbar ist. Denn:

  * Die Rahmenabbildung Omega --> Rad(A), phi |-> { x | x nilp. oder phi }
    ist genau dann ein Iso, wenn sie bijektiv ist.

  * Injektiv ist sie stets.

  * Surjektiv ist sie genau dann, wenn für jedes Radikalideal J gilt:
    forall x \in J: x nilp. oder 1 \in J.

  * Das ist genau dann der Fall, wenn für jedes f aus A gilt:
    f nilp. oder f invertierbar.
    (Betrachte für eine Richtung f \in sqrt(f)).

* O_X erfüllt die Bedingung "nilpotent oder invertierbar" im Allgemeinen
  nicht. Betrachte etwa X = Spec Z und f = 2. Da O_X reduziert ist, ist "2
  nilpotent" äquivalent zu "2 = 0". Es gilt aber "nicht (2 = 0)". Invertierbar
  ist 2 aber auch nicht. [Kürzer: Die Aussage ist geometrisch. Im Halm zum
  Primideal (2) ist 2 weder nilpotent noch invertierbar.]


=== Wann ist das (volle Radikalideal-)Spektrum boolsch?

* Ist a ein Radikalideal, so gilt

      neg a = { x | xu nilpotent für alle u in a }.

* Ein Radikalideal a ist genau dann dicht, wenn

      a cap b = sqrt(0) ==> b = sqrt(0)

  für alle Radikalideale b. Man kann auch nur Ideale der Form sqrt(h) nehmen.

* Spec(A) ist boolsch genau dann, wenn das Einsideal das einzige
  dichte Ideal ist. In diesem Fall gilt dim A <= 0. Die Umkehrung gilt
  vielleicht auch...

* Ein reduzierter Ring ist genau dann von Dimension <= 0, wenn er ein
  von-Neumann-regulärer Ring ist. Das bedeutet, dass für alle x gilt:
  x^2 | x.


=== Zusammenhang internes Spektrum und relatives Spektrum

Wenn man davon ausgeht, dass die universelle Eigenschaft des relativen
Spektrums sogar in der Kategorie der lokal geringten Örtlichkeiten gilt, dann
hat man:

    Hom_{LRL/(X,O_X)}(W, RelSpec A) ~~
    Hom_{LRL(Sh(X))/(pt,O_X)}(W, RelSpec A) ~~
    Hom_{O_X}(A, mu_* O_W).

Das interne Spektrum erfüllt eine ähnliche, aber nicht gleiche universelle
Eigenschaft (zumindest ist das sehr plausibel -- natürlich reden die Leute
nicht über lokal geringte Örtlichkeiten):

    Hom_{LRL(Sh(X))/(IntSpec Z)}(W, IntSpec A) ~~
    Hom_Ring(A, mu_* O_W)

    Hom_{LRL(Sh(X))/(IntSpec O_X)}(W, IntSpec A) ~~
    Hom_{O_X}(A, mu_* O_W)

Daher sieht man: Wenn IntSpec O_X ~~ (pt, O_X) sein sollte, dann würde IntSpec A
die universelle Eigenschaft von RelSpec A erfüllen.

Es stimmt im Allgemeinen aber nicht, dass IntSpec O_X ~~ (pt, O_X).

* Spec ist also ein guter Funktor Ring^op --> LRL/(Spec Z).
  Hier ist aber ein guter Funktor Alg(R)^op --> LRL/(pt, R) gesucht, wobei
  R ein lokaler Ring ist (der im Anwendungsfall O_X noch die Voraussetzung
  "nicht invertierbar ==> nilpotent" erfüllt).

* Ein Vorschlag wäre, Spec R' := Spec Reg^(-1)R zu setzen.
  Dieses Spektrum ist genau dann einpunktig, wenn jedes Element aus R
  nilpotent oder regulär ist.

  Für R = O_X bedeutet diese Bedingung insbesondere, dass für jede globale
  Funktion f gilt:

      X = Nilp(f) cup Reg(f),

  wobei Nilp(f) = { x | f nilpotent in O_{X,x} } und
  Reg(f) = { x | f regulär in allen Halmen auf einer Umgebung um x }.
  (So definiert sind beide Mengen offen, und disjunkt voneinander.)

  Für R = O_X zu X = Spec A, A Integritätsbereich, ist diese Bedingung erfüllt!
  Sogar trivialerweise, zu Überdeckungen muss man nicht übergehen.

  Für R = O_X zu X = Spec k[x,y]/(xy) aber nicht. Gegenbeispiel ist die
  Funktion x im lokalen Ring beim Ursprung. Diese ist nicht nilpotent und auch
  nicht regulär dort, insbesondere also nicht auf einer kleinen Umgebung.

* Bemerkung: Die Betrachtung von Spec (R/m), m das Ideal der nilpotenten bzw.
  nicht-invertierbaren Elemente, hilft nicht. Denn Rad(R) = Rad(R/m).
  R/m ist nur ein Körper im Sinne von "nicht invertierbar ==> null".

* Ein anderer Vorschlag wäre, als neues Spektrum nicht die Menge aller
  Radikalideale zu nehmen, sondern nur solche, sodass für jedes Element x von R,
  dessen Bild im gegebenen Radikalideal liegt, x nilpotent oder dieses Ideal
  schon das Einsideal ist.

  Diese Bedingung ist stabil unter Schnittbildung, aber nicht unter
  Summenbildung. Man kann eine Art Reflektor definieren, über

      Ja := bigcap_{a \subseteq b, b erfüllt Bedingung} b.

  Aber dann konnte ich nicht J(a cap a') = Ja cap Ja' zeigen. Das bräuchte man
  aber, wenn man damit eine Unterörtlichkeit definieren möchte.

  Ob damit die Abbildung Omega --> Rad' surjektiv wird, ist auch nicht ganz
  klar.

* Hakims Spec-Konstruktion RL --> LRL ist zwar ein Adjungierter zum
  Vergissfunktor, aber nicht volltreu. Es gilt zwar

      Hom_{LRT/E}(Spec B, Spec A) = Hom_{RT/E}(Spec B, A) = Hom(A, B)

  für Ringe in einem Topos E; aber Hom_LRT(Spec B, Spec A) könnte viel größer
  als Hom(A, B) sein. Treu ist der Funktor wohl schon.

  Siehe Kommentar von Zhen Lin bei
  http://math.stackexchange.com/questions/1532244/joyals-construction-of-the-spectrum-of-a-commutative-ring?noredirect=1#comment3122269_1532244.

=== Internes Spektrum, "richtig"

Beobachtung: Die Menge der *quasikohärenten* Radikalideale von O_X entspricht
extern der Menge der offenen Teilmengen von X:

    quasikohärentes Radikalideal I
        |--> offene Menge { x \in X | 1 \in I_x }

    offene Menge U
        |--> Idealgarbe zum reduzierten abgeschlossenen Unterschema X \ U

* Da es vermutlich einen geeigneten Reflektor Radikalideale --> quasikohärente
  Radikalideale gibt, wird Rad_qcoh(O_X) zu einem internen Rahmen.
  ...aber der Reflektor ist ein *Links*adjungierter!

  Update: Es passt wunderbar, dass der Reflektor ein Linksadjungierter ist:
  a,b <= a+b <= (a+b)#.
  Sei a,b <= i. Dann a+b <= i. Dann (a+b)# <= i. Gut!
  Garbifizierung ist doch auch ein Linksadjungierter!

* Analog könnte man Rad_qcoh(A) und so Spec_int(A) definieren.
  Erfüllt das die richtigen Eigenschaften?

* Für das so definierte interne Spektrum gilt:

      Spec_int(O_X[f^(-1)]) ~~
          Ouv(D(f)) = { phi : Omega | (f inv. ==> phi) ==> phi }

      Spec_int(O_X/(f)) ~~
          Ouv(V(f)) = { phi : Omega | (f inv. vee phi) <==> phi }
                    = { phi : Omega | f inv. ==> phi }

  Die Isos sehen jeweils so aus: J |-> (1 in J), phi |-> bigvee { (1) | phi }.
  Konkret ist das zugehörige pseudokohärente Radikalideal zu einem
  Wahrheitswert phi im ersten Fall

      J = { x in O_X[f^(-1)] | x inv. ==> phi }

  und im zweiten Fall

      J = { x in O_X/(f) | x inv. ==> (f inv. vee phi) }.

  Da ([h] inv. <==> h inv. vee f inv.), spielt es keine Rolle, ob hier "x inv."
  in O_X/(f) oder in O_X gelesen wird.

* Das pseudokohärente Spektrum ist im Allgemeinen nicht kompakt. Es ist
  auch Spec_X(A) --> X im Allgemeinen nicht topologisch eigentlich. Das
  würde das ja bedeuten. (Das grundlegende Problem ist: Die gerichtete
  Vereinigung von pseudokohärenten Radikalidealen ist zwar wieder ein
  Radikalideal, aber nicht unbedingt gerichtet.)

  Aber es gilt: Sei I eine (externe) gerichtete Menge. Sei A eine O_X-Algebra
  über einem Schema. Wird dann durch Spec_rel(A) durch eine I-indizierte
  Familie von Offenen überdeckt, so genügt schon ein einzelnes Offenes.

  Das zeigt man mit einem Induktionsbeweis der Behauptung

      forall m. forall h:O. forall s:A.
          (h inv. ==> s in J^m) ==> hs in J_alpha für ein alpha.

  Dabei Coh(bigcup_alpha J_alpha) =: bigcup_{m >= 0} J^m.

* Hier eine völlig explizite Begründung, dass Spec^rel im Allgemeinen nicht
  kompakt ist.

  1. Kompakte Örtlichkeiten haben auch die Frobeniusreziprozitätseigenschaft.
  2. Spec^rel(O_X[f^(-1)]) ist isomorph zu { phi : Omega | phi ==> f inv. }.
  3. Diese Örtlichkeit besitzt die Frobeniusreziprozitätseigenschaft genau dann,
     wenn f nilpotent oder invertierbar ist.

  Die Frobeniusreziprozitätsbedingung für { phi : Omega | phi ==> f inv. }
  besagt:

      forall phi mit (phi ==> f inv.). forall psi.
          (f inv. ==> phi v psi) ==> ((f inv. ==> phi) v psi).

  Dazu äquivalent ist:

      forall phi, psi.
          (f inv. ==> (phi wedge f inv.) v psi) ==> ((f inv. ==> (phi wedge f inv.)) v psi).

  Das wiederum ist äquivalent zu:

      forall phi, psi.
          (f inv. ==> phi v psi) ==> ((f inv. ==> phi) v psi).

  Falls f nilpotent oder invertierbar ist, gilt das (hier geht 0 != 1 in O_X
  ein). Umgekehrt erhält man gerade diese Bedingung durch die Instanz

      phi := bot,  psi := (f inv.)

  (hier geht die Beinahe-Körpereigenschaft von O_X ein). (Über beliebigen LRL
  kann es also noch allgemeinere Möglichkeiten geben, dass dieses Spektrum die
  Frobeniusreziprozitätseigenschaft hat.)

* Da das pseudokohärente Spektrum im Allgemeinen nicht kompakt ist, ist klar:
  Es gibt keine Möglichkeit, dass pseudokohärente Spektrum als ein gewöhnliches
  anzusehen.

* Spec^rel ist eine Unterörtlichkeit von Spec. Und zwar die, wo die Abbildung
  \underline{O_X} --> O_{Spec^rel A} lokal ist. Denn an einem Halm F (der zu
  einem Filter von A gehört) ist ja O_X --> A[F^(-1)]. Und diese Abbildung ist
  halt genau dann lokal, wenn für F gilt: Sollte phi(f) in F liegen, so ist f
  in O_X invertierbar.

* Kann wohl zeigen:

      Hom_LRT(X, Spec A) = Hom_RT(X, (Set,A))

  für alle lokal geringten Grothendiecktopoi X.

  Denn:

      Hom_LRT(X, Spec A) = 
      { (X --f--> Spec A, f^{-1}(O_{Spec A}) --> O_X lokal) } =
      { (F <= AA auf X Filter, AA[F^(-1)] --> O_X lokal) } =
      { AA --> O_X beliebiger Ringhomo }.

  Genauso für eine k-Algebra A:

      Hom_{LRT/(Set,k)}(X, Spec(A|k)) = Hom_{RT/(Set,k)}(X, (Set,A)}

  für alle lokal geringten Grothendiecktopoi X, für die k --> O_X lokal ist.
  Die Notation "LRT/(Set,k)" ist abuse of notation, da ich nicht unbedingt
  voraussetzen möchte, dass k selbst ein lokaler Ring ist. Aber ist er's nicht,
  so sind beide Seiten nur für X = {} nicht leer, oder?

  Kann das alles pimpen zu einer relativen Situation. Nutze

      Hom_{LRT/(E,O_E)}(X, Spec(A|O_E)) =
        Hom_{LRT(E)/(pt,O_E)}(XX, Spec(A|O_E))

  und

      Hom_{RT/(E,O_E)}(X, (E,A)) =
        Hom_{RT(E)/(pt,O_E)}(XX, Spec(A|O_E)).

  Dabei ist XX die Auffassung von X als lokal-interne Kategorie in E.

  Als Korollar von dem ganzen sieht man, dass Monique Hakims Spektrum lokalisch
  ist.

  Außerdem: Sei F : I --> LRL ein Diagramm. Dann gilt

      lim F = Spec(O_{lim^RL F} | F(i), i in I).

  Was bedeutet das kategoriell?


=== Interne Charakterisierung des modalen Operators fürs pseudokohärente Spektrum

* Sei A eine k-Algebra. Sei k ein lokaler Ring. (Zum Beispiel die Strukturgarbe
  eines Schemas.) In Spec(A) gibt es ja einen modalen Operator Box, der
  Spec(A|k) ausschneidet (also das richtige relative Spektrum). Kann man den
  in der internen Sprache von Spec(A) charakterisieren?

  Ein Vorschlag lautet:

      Box(phi)  <==>  1 in Coh{1|phi} <= A_. (Oder k_?)

* Eine verwandte Frage ist: Sei A eine k-Algebra. Gibt es einen modalen
  Operator Box, sodass für Radikalideale J von A gilt:?

      1 in Coh(J)   <==>   Box(1 in J).

  Die Antwort lautet "nein". Denn: Durch Betrachtung von

      J := { x | x nilpotent oder phi }

  erhält man, dass Box(phi) äquivalent zu (1 in sup { (1) | phi }) ist
  (mit dem Supremum in Rad(A)). Nun gilt zwar

      1 in Coh(J)  <==  (1) = sup { (1) | 1 in J },

  aber die Rückrichtung gilt wohl nicht. Eine wichtige Ausnahme ist der
  Fall A = k, bei dem genau dann 1 in Coh(J), wenn 1 in J.


=== Spektrum als universelle Lokalisierung

* Universelle Eigenschaft von Spec(A): A --> O_{Spec(A)} ist
  die universelle Lokalisierung von A. (Diese Eigenschaft spielt sich in LRL^op
  oder LRT^op ab.)

* Universelle Eigenschaft von RelSpec_{O_X}(A): A --> O_{RelSpec}
  ist die universelle Lokalisierung von A mit der Nebenbedingung, dass
  O_X --> A --> O_RelSpec lokal ist.

* Achtung: Sei B ein lokaler Ring in Spec(A) zusammen mit einem Homo
  AA --> B. Dann stimmt es im Allgemeinen *nicht*, dass es einen lokalen Homo
  O_{Spec A} --> B in Spec(A) gibt. Im Allgemein muss man noch mit einer
  geeigneter topologischen Abbildung zurückziehen.

* Witzig: A --> O_{Spec(A)} ist sogar eine Lokalisierung (nämlich Invertierung
  der Elemente im generischen Filter), obwohl man das nicht vorab gefordert hat.


=== Pseudokohärenz

Sei k ein Ring. Ein Radikalideal I in einer k-Algebra A heißt genau dann
pseudokohärent, wenn

    forall f:k. forall s:A. (f inv. in k ==> s in I) ==> fs in I.

* Sei I ein pseudokohärentes Radikalideal in einer k-Algebra A.
  Dann ist auch das Urbild von A in k pseudokohärent.

  Allgemeiner: Sei phi : A --> B ein Morphismus von k-Algebren.
  Sei I <= B ein pseudokohärentes Radikalideal. Dann ist auch phi^(-1)[I]
  pseudokohärent. Denn:

      Sei f : k. Sei s : A. Gelte (f inv. ==> phi(s) in I).
      Dann folgt f^n phi(s) in I, also phi(f^n s) in I.
      Also f^n s in phi^(-1)[I].

* Sei k --> A ein lokaler Morphismus lokaler Ringe. Sei A *quasikohärent*
  über A. Dann ist A auch quasikohärent über k. (Dabei geht ein, dass
  A[f^(-1)] ~~ A[phi(f)^{-1}] für f aus k.)

* Gibt es eine Klasse von Morphismen, sodass ein Ideal I <= A genau dann
  pseudokohärent ist, wenn

      Hom(?, I) --> Hom(?, I) ==> Hom(?, I)

  für alle Morphismen aus der Klasse ein Differenzkerndiagramm ist?

  Oder vielleicht mit "? tensor I" statt "Hom(?, I)"?

* Sei A eine quasikohärente O_X-Algebrengarbe in dem kleinen Zariski-Topos
  eines Schema. Dann ist sqrt(0) <= A schon pseudokohärent.

  Folgerung: Man könnte denken, dass die Äquivalenz
  
      I <= A pseudokohärent  <==>  Coh(sqrt(0)) <= I

  gilt. Die Richtung "==>" ist auch sicher erfüllt. Aber "<==" ist im
  Allgemeinen falsch, wie schon das Beispiel A = O_X (in einem Schema) zeigt.

* Sei A quasikohärent auf einem Schema (vielleicht geht's auch allgemeiner).
  Dann ist A Box-separiert, wobei

      Box phi = (1 in Coh { 1 | phi } <= A).

  Zeige dazu für s:A mit Induktion:

      forall n. forall x:A. x in J_n ==> exists m. x^m s = 0.

  Dabei ist J_n aus der Konstruktion von Coh { 1 | s = 0 }. Der Beweis geht so:

  * Induktionsanfang n = 0: x ist nilpotent oder s = 0.
  * Induktionsschritt: Sei x^l = sum_i f_i s_i mit (f_i inv ==> s_i in
    (vorher)). Also (f_i inv ==> bigvee_{l_i} s_i^{l_i} s = 0).
    Diese Vereinigung ist gerichtet, also können wir das bigvee nach vorne
    ziehen. Da außerdem A quasikohärent ist, folgt: f_i^{m_i} s_i^{l_i} s = 0.
    Sei N das Maximum aller m_i und l_i. Eine hinreichend hohe Potenz von x ist
    dann von der Form sum_i ??_i (f_i s_i)^N. Diese annihiliert s.

  Vielleicht ist ein Modul genau dann quasikohärent, wenn er eine Box-Garbe
  ist. Das habe ich nicht mehr weiter untersucht.

=== Sichtpunkt von klassifizierendenden Topoi

* Die Angabe eines Filters in A ist gleichbedeutend zur Angabe
  einer Lokalisierung von A bis auf Isomorphie.

* Sei f : Zar(A) --> Spec(A) durch die Lokalisierung O_{Spec A}~ von A
  (also T -mu-> Spec(A) |-> Gamma(mu^(-1) O_{Spec A})) gegeben.

* Sei g : Spec(A) --> Zar(A) durch die lokale A-Algebra O_{Spec A} gegeben.

* Dann ist (f^* . g^*)(A^1) = f^*(O_{Spec A}) = O_{Spec A}~.


=== Pseudokohärentes Spektrum als gewöhnliches Spektrum

* Es kann (im Allgemeinen) kein Ideal J_0 geben, sodass für alle
  Radikalideale J <= A gilt: J ist genau dann pseudokohärent, wenn J_0 <= J.

  Denn: Wenn es ein solches Ideal J_0 gibt, dann tut's sqrt(J_0) auch. Also
  können wir ohne Einschränkung davon ausgehen, dass J_0 ein Radikalideal ist;
  somit sqrt(0) <= J_0. Andererseits folgt J_0 <= Coh(0).

  Im Spezialfall, dass A eine quasikohärente O_X-Algebra auf einem Schema ist,
  ist Coh(0) = sqrt(0), also J_0 = sqrt(0). Von diesem J_0 wissen wir aber,
  dass es die gewünschte Bedingung nicht erfüllt (denke A = O_X, X nicht
  nulldimensional).
